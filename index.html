How to Install Soft Vortex

  1. Download the Soft Vortex NBT by clicking the download button.
  2. Press Win + R and type %appdata%.
  3. Locate the .minecraft folder and double-click it.
  4. Replace the hotbar.nbt file with the new downloaded version.
